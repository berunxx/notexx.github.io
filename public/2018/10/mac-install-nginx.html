Mac对nginx 启动、关闭、重启

发表于 2018-10-22   |   更新于 2018-10-27 | 分类于 Mac |
字数统计: 76 | 阅读时长 ≈ 1

问题

在终端中执行：
brew install nginx
安装完成之后，nginx配置文件路径在/usr/local/etc/nginx下

常用命令

查看nginx版本
nginx -v

启动nginx：brew services start nginx

关闭nginx：brew services stop nginx

也可以使用如下快捷指令
重新启动nginx：
nginx -s reload
停止nginx：
nginx -s stop
